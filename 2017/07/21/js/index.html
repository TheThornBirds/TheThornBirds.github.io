JavaScript的对象是一组由键-值组成的无序集合，JavaScript对象的键都是字符串类型，值可以是任意数据类型，要获取一个对象的属性，我们用对象变量.属性名的方式

9.变量

申明一个变量用var语句

var a; // 申明了变量a，此时a的值为undefined
var $b = 1; // 申明了变量$b，同时给$b赋值，此时$b的值为1
var s_007 = '007'; // s_007是一个字符串
var Answer = true; // Answer是一个布尔值true
var t = null; // t的值是null

变量名也可以用中文

同一个变量可以反复赋值，而且可以是不同类型的变量

字符串

1.js的字符串就是用’’或””括起来的字符表示，遇到特殊字符可使用\转义

2.多行字符串，使用...表示，反引号就是小波浪键

把多个字符串连接起来，可以用+号连接，例如：

var name = 'xiaoming';
var age = 20;
var message = 'nihao,'+name+'ni jinnian'+age+'sui le !'

3.模版字符串，ES6新增了一种模板字符串，不用使用+号，改为${}

var name = 'xiaoming';
var age = 20;
var message = 'nihao,${name},nijinnian${age}sui le !'

4.操作字符串
字符串.length 获取字符串长度

获取指定位置的字符，使用下标操作，超出索引一般不会报错，一律返回undefined

字符串不可变
