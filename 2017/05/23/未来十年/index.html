未来十年

发表于 2017-05-23 | | 阅读次数

过去未去，未来已来。

从我说起

　　我叫吴晨，生于1995年，人生二十来年，有起有伏。但无论得失，都已经是过去的东西。2016年三月起开始接触编程，犹记得当时满腔热血，定要转行。我在网易云课堂上自学了C语言的一些基础。第一次用C语言写出九九乘法表的时候开心的整夜睡不着。感谢翁恺老师的课。让我从那时起有了未来奋斗的目标。

过去

　　16年六月份我来到北京。当时是报着来学习计算机的心情来的，可惜中间发生了很多这样那样的事情。我也尝试跑了很远，找一些计算机相关的工作，结果并不如人愿。家里很反对我走计算机这条路。因为我大学学了三年的外贸。但是我觉得，人生短短几十年。我连睡觉都觉得浪费时间，怎么能花时间走一条自己并不喜欢的路呢？我要走肯定就是一条道走到黑了。所以，这条路，必须是我喜欢，并且感兴趣的一条路。因为我的梦想，就是在这一生中在某一个领域，有所成就。

　　于是当年九月，我在一个培训班贷款报名学习了java，从九月学到17年三月份，期间很感谢我那位老师。他带我走上了编程的道路。学习了基础语法，学习了javaEE，学习了一些框架。我自己也学习了很多东西，半年中收获极大。我很开心。

　　2017年4月11日，我收到了朗姿的入职通知，工作是做java开发。当时的心情真的欣喜若狂，难以言表。因为我找了将近一个多月的工作，深知其中艰辛。非常感谢公司不嫌弃我的学历，不嫌弃我的专业，不嫌弃我没有工作经验。我于4月17日正式入职了。

未来

　　我真的还有好长的路要走。特在此做出十年规划。

  • 17年年底，我要熟悉我所有的工作流程。工作之余学习设计模式。

  • 两年内，我必须掌握两门以上的编程语言。并还光所有欠下的债务。

  • 三年内，我要通过自己的努力，过上自己想要的生活，并尝试买一辆汽车。

  • 五年内，我要让自己和女朋友花钱的时候可以不用考虑太多，实现财务自由。

  • 六年内，我要娶到我心爱的刘先艳，跟她结婚，跟她相守。

  • 七年内，我应该会有自己的宝宝。并开始着手买房相关事宜。

  • 八年内，我要有一个自己的窝。这些事情我一定要在我三十岁之前完成。

  • 十年内，我希望能够有自己的公司，或者成为一家公司不可或缺的人物，并为自己的事业，一直努力下去。

过去未去，未来已来。2027，见真章。

ps:很久不写东西了，我以后会尽力做到一个月更新一篇博客。加油。

吴晨 2017年5月23日 晴
