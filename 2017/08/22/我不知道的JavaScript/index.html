1.== 和 ===的区别

==比较，它会自动转换数据类型再比较，很多时候，会得到非常诡异的结果；

===比较，它不会自动转换数据类型，如果数据类型不一致，返回false，如果一致，再比较。

所以应该坚持使用 === 比较

3.数组操作小技巧

indexOf() 搜索一个指定元素的位置

slice() 截取数组的部分元素，返回一个新的数组,参数包头不包尾

push() 向数组的末尾添加若干元素

pop() 删除数组的最后一个元素

unshift() 向数组头部添加若干元素

shift() 删除数组的第一个元素

sort()　对数组进行排序，会直接修改当前数组的元素位置

reverse() 数组元素反转，123变成321

splice() 从制定的所以开始删除若干元素，再从改位置添加若干元素

concat() 连接两个数组，返回一个新的数组

join() 把当前数组的每个元素都用指定的字符串连接起来，返回连接后的字符串

4.for … in 和 for … of 的区别

5.函数的定义

function abs(x) {
    if (x >= 0) {
        return x;
    } else {
        return -x;
    }
}
  • function指出这是一个函数定义；
  • abs是函数的名称；
  • (x)括号内列出函数的参数，多个参数以，分隔
  • { ... }之间的代码是函数体，包含若干语句

实际上上述定义的abs()也是一个函数对象，函数名abs视为指向该函数的变量。

rest参数可以获得已定义参数之外的参数

6.函数变量作用域

js的函数会先扫描整个函数体，并把所有的变量提升到函数顶部，所以我们应该严格遵守”在函数内部首先申明所有变量”的原则，最常见的做法是用一个var申明函数内部用到的所有变量

7.全局作用域

不在任何函数内部定义的变量就具备全局作用域，实际上全局作用域的对象全部都被绑定到了Window的属性上，这说明JavaScript实际上只有一个全局作用域。任何变量（函数也视为变量），如果没有在当前函数作用域中找到，就会继续往上查找，最后如果在全局作用域中也没有找到，则报ReferenceError错误。

8.解决块级作用域变量的问题

for循环语句块中定义的变量依旧可以拿到语句外引用，为了解决这一问题，ES6引用关键字let，用let替代var可以申明一个块级作用域的变量。ES6同样也引用了新的关键字const来定义常量。

9.this的指向问题

如果以对象的方法形式调用，该函数的this指向被调用的对象。如果单独调用函数，该函数的this指向全局对象，也就是window。
